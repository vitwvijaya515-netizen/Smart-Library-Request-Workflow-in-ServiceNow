# Smart-Library-Request-Workflow-in-ServiceNow
This project involves developing a role-based Library Management System on the ServiceNow platform to automate and streamline library operations in an academic environment. The system enables students to request books and track request status, while librarians manage the book inventory and control the approval process. Role-based access controls ensure secure operations, automated workflows handle request processing and availability updates, and reporting features provide insights into borrowing trends. The solution improves visibility, efficiency, and accuracy in managing library resources
